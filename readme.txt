Проект содержит два теста
- checkYandexTitle():
-- переходит на страницу yandex.ru в браузере, открытом на весь экран
-- проверяет title страницы

- checkTelephoneSearchOnTele2Page():
-- открывает страницу tele2
-- вводит в строку поиска текст "97"
-- проверяет, что в первом телефоне в результатах поиска присутствуют цифры "9" и "7"

Во втором тесте используются явные ожидания для появления поля ввода и исчезновения спиннера загрузки.

Проект использует: Selenium Java, WebDriverManager, jUnit, log4j.
